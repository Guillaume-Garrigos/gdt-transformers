Cette page regroupe les informations concernant le [GdT Transformers](https://guillaume-garrigos.com/gdt-transformers/) organisé au LPSM. Les exposés ont lieu au Bâtiment Sophie Germain à l'Université Paris Cité (Campus des Grands Moulins, Place Aurélie Nemours dans le 13e). Les dates et horaires seront amenés à varier.

Organisateurs pour l'année 2025-2026: Simon Coste, Guillaume Garrigos, Tam Le.

Vous pouvez vous abonner au calendrier du GdT en intégrant [ce fichier .ics](https://calendar.google.com/calendar/ical/bf059be5a5fd6243b028026af74ae13fedb87e6a3f12d82179b38237f47422ef%40group.calendar.google.com/public/basic.ics) à votre calendrier.
